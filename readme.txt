I used RPGViewer30Build20240319 to extract the GossipText.ini, SystemText.ini, TermText.ini files from Patch.PCK file

Download link I used [https://www.mefcl.com/rpgviewer.html]

File > Open > HongKong&Taiwan > Odin > Heros of the Three Kingdoms series > Heros of the three kingdoms 7 > [Pick your game directory]

[Select Patch.PCK] > Addon > Archive > Extract

To use the translations files, in the game directory create a folder called Setting then paste the three ini files into it

These translations are my own personal translation preferences where only important things are translated and many texts are kept in the original language for best UI and experience.
Main things that are translated are ability/item/skill descriptions etc.

Thanks to Togari whose translations I referenced in order to create my own version.
